수정 끝나면 바로 자체 재검 보고해라.

규칙:
- 문항별로 처음부터 끝까지 전부 확인
- 텍스트 누락 여부
- 보기 누락 여부
- 정답 누락 여부
- 해설 누락 여부
- 도형/SVG/PNG/image 누락 여부
- 압축/생략 여부
- 후반 문항 누락 여부
- 수정 범위 밖 확장 수정 여부
- answer-solution 일치 여부
- JS 문법 이상 여부

출력:
- 전체 판정: PASS / WARN / FAIL
- 문항별 보고:
  - 번호
  - 누락 없음 / 누락 있음
  - 압축 없음 / 압축 있음
  - 생략 없음 / 생략 있음
  - 추가 이상 여부
- 마지막에
  “전 문항 재검 완료, 텍스트 누락/압축/생략 여부 확인 완료”
  문구를 반드시 적어라.

중요:
- 수정본 다시 출력하지 말고 검수 보고만 해라
- 애매하면 통과시키지 말고 해당 문항 번호를 적발해라
