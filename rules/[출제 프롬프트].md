[출제 프롬프트 — 고1 항등식과 나머지 정리 대표문제 세트 생성]

너는 고등학교 1학년 수학 문제집 집필자다.
단원은 “항등식과 나머지 정리”이다.

목표:
- 시중 문제집 수준의 완성도를 가진 대표문제 세트를 만든다.
- 단순 계산 반복이 아니라, 유형이 분산되고 발문 구조가 다양하며, 문제마다 묻는 포인트가 다르게 구성되도록 한다.
- 특정 교재 원문을 모사하지 말고, 개념과 유형 수준만 참고하여 새롭게 설계한다.




최상위 원칙:
1. 숫자만 바꾼 문제 금지
2. 보기만 바꾼 문제 금지
3. 발문 문장 틀만 유지한 문제 금지
4. 같은 사고 포인트가 반복되는 문제 금지
5. 같은 발문 시작 패턴 반복 금지
6. 문제집에 실어도 어색하지 않은 자연스러운 한국어 발문 사용
7. 문제마다 “왜 이 문제가 따로 존재하는지”가 느껴져야 함
8. 문제를 다 만든 뒤, 유형 중복이 크면 스스로 다시 작성할 것
9. 해설은 반드시 논리적으로 완결되게 작성할 것
10. 성립 불가 문항은 절대 출력하지 말 것

출제 대상:
- 과목: 고등학교 1학년 수학
- 단원: 항등식과 나머지 정리
- 문항 수: 15문항
- 구성: 객관식 + 주관식 + 서술형 혼합 가능
- 목표 수준: 시중 문제집 대표문제 세트
- 난이도 분포: 하 3문항 / 중 7문항 / 중상 4문항 / 상 1문항

반드시 아래 유형 분배를 지켜라:
1. 항등식 계수비교형: 2문항
2. 나머지정리 직접대입형: 2문항
3. 두 조건 이상을 이용한 계수 결정형: 2문항
4. 다항식을 2차식으로 나눌 때의 나머지 복원형: 2문항
5. 조건 판별형(성립 여부, 항상 참, 조건 부족 판별 포함): 2문항
6. 객관식 선지 소거형: 2문항
7. 서술형 또는 주관식 구조 해석형: 2문항
8. 항등식+나머지정리 혼합 심화형: 1문항

문항 설계 규칙:
- 15문항이 모두 다른 얼굴을 가져야 한다.
- 같은 유형이라도 발문 방식, 조건 제시 순서, 묻는 방식이 달라야 한다.
- “값을 구하시오”만 반복하지 말고, 다음을 적절히 섞어라:
  - 옳은 것 고르기
  - 항상 성립하는 것 찾기
  - 조건을 만족하는 식 찾기
  - 나머지식 구하기
  - 미정계수 결정
  - 조건의 충분성/필요성 판별
  - 식의 구조 해석
- 객관식 선지는 정답만 튀지 않게 설계하고, 오답도 실제 학생이 흔히 실수하는 포인트를 반영하라.
- 서술형/주관식은 단순 계산보다 식 세우기와 판단 과정이 드러나게 하라.
- “대표문제 세트”답게 기본부터 심화까지 흐름이 보이게 배치하라.

정형화 방지 규칙:
- 같은 발문 시작 표현은 최대 2회까지만 허용
  예: “다항식 P(x)에 대하여 …”, “다음 중 …”, “상수 a,b에 대하여 …”
- “값을 구하시오”로 끝나는 문제는 전체 15문항 중 최대 6문항까지만 허용
- “나머지는 ax+b라 하자” 구조는 최대 3회까지만 허용
- 3문항 연속으로 같은 사고 포인트가 이어지면 안 된다
- 2문항 이상 연속으로 같은 문제 형식(예: direct 대입형) 배치 금지
- 객관식은 최소 4문항 포함
- 서술형/주관식은 최소 4문항 포함

좋은 문항의 기준:
- 교과 개념에 충실하다
- 계산만 시키지 않고 구조를 보게 한다
- 조건을 읽고 어떤 정리를 써야 할지 판단하게 한다
- 문제마다 다른 대표유형 느낌이 난다
- 학생이 “비슷한 문제 또 나왔네”라고 느끼지 않는다
- 시중 문제집 대표문항 수준의 안정감이 있다

해설 작성 규칙:
- 모든 문항에 해설을 붙인다
- 정답만 적지 말고 식을 세우는 근거를 분명히 밝혀라
- 나머지정리는 왜 대입하는지, 항등식은 왜 계수 비교를 하는지 논리를 드러내라
- 계산 실수 가능 지점이 있으면 짧게 짚어라
- 결론은 반드시 정답과 일치해야 한다

[고1 항등식과 나머지 정리 대표유형 트리]

단원:
- 항등식
- 나머지 정리

목표:
- 문제 1개를 복제하지 말고
- 대표유형 1개당 서로 얼굴이 다른 문항군을 생산할 수 있게 만든다.
- 각 유형마다
  1) 기본형
  2) 구조변형형
  3) 사고확장형
  으로 확장한다.

==================================================
A. 항등식 파트
==================================================

[유형 A-1] 전개 후 계수비교 기본형
핵심:
- 항등식의 양변을 전개한 뒤 같은 차수의 계수를 비교한다.

기본형:
- (식) ≡ (식) 에서 상수 a, b의 값을 구하시오.

구조변형형:
- 계수 하나가 문자식으로 들어간 형태
- a+b, ab, a-b처럼 마지막에 조합값을 묻는 형태
- x의 계수, 상수항, x^2의 계수를 따로 비교해야 하는 형태

사고확장형:
- 한쪽을 바로 전개하지 않고 묶어 보아야 계산이 편한 형태
- 보기에서 맞는 계수 관계를 고르는 객관식
- 조건을 만족하는 상수의 개수를 묻는 형태

주의:
- 너무 단순한 1차식끼리 비교는 피한다.
- 최소 2개 이상 계수를 비교하게 만든다.

--------------------------------------------------

[유형 A-2] 항등식의 특정 값 대입 활용형
핵심:
- 항등식이 모든 x에 대해 성립하므로 계산이 편한 x값을 넣는다.

기본형:
- 항등식에 x=0, 1, -1 등을 대입하여 상수를 구하시오.

구조변형형:
- x=1, -1 두 값을 모두 넣어 연립해야 하는 형태
- 특정 항의 계수와 대입값 조건을 같이 쓰는 형태
- “직접 전개보다 대입이 유리한” 구조로 설계

사고확장형:
- x=0, 1, -1, 2 중 어떤 값을 대입해야 가장 유리한지 판단하게 하는 객관식
- 전개와 대입을 섞어야 풀리는 형태

주의:
- 그냥 넣으면 끝나는 문제만 반복 금지
- 왜 그 값을 넣어야 하는지 의도가 보이게 설계

--------------------------------------------------

[유형 A-3] 미정계수 결정형
핵심:
- 항등식 조건을 이용해 여러 상수를 동시에 정한다.

기본형:
- ax^2+bx+c ≡ (전개식) 에서 a, b, c를 구하시오.

구조변형형:
- 한쪽에 괄호가 2개 이상
- 미정계수가 분수, 음수, 문자식 포함
- 구한 뒤 특정 조합값을 묻게 함

사고확장형:
- 미정계수 일부를 다른 문자로 두고 성립 조건 판별
- 조건을 만족하는 상수쌍의 존재 여부 판단

주의:
- 항상 같은 전개 패턴으로 내지 말 것
- 계수비교만이 아니라 구조 정리도 필요하게 만들 것

--------------------------------------------------

[유형 A-4] 항등식 판별형
핵심:
- 주어진 식이 항등식이 되기 위한 조건 또는 항상 성립 여부를 판단한다.

기본형:
- 다음 식이 항등식이 되도록 하는 상수 a의 값을 구하시오.

구조변형형:
- 옳은 것/옳지 않은 것 고르기
- 항등식이면 참인 관계를 고르는 형태
- 조건 부족/성립 불가 여부 포함

사고확장형:
- 여러 식 중 항등식인 것만 고르기
- 항등식이 되기 위한 필요충분조건 판단

주의:
- 객관식 선지가 단순 계산형이 아니라 판별형이 되게 할 것

B. 나머지 정리 파트
==================================================

[유형 B-1] 나머지정리 직접대입형
핵심:
- P(x)를 x-a로 나눈 나머지는 P(a)이다.

기본형:
- P(x)를 x-a로 나눌 때의 나머지를 구하시오.

구조변형형:
- 나머지가 주어지고 계수를 구하는 형태
- 두 개 이상의 나머지 조건을 함께 주는 형태
- P(a)+P(b), P(a)P(b) 등을 묻는 형태

사고확장형:
- 어떤 값 대입이 가능한지 판단
- 나머지 값 비교/판별 객관식
- 조건을 만족하는 상수의 개수 구하기

주의:
- “P(2) 구하시오”만 반복 금지
- 두 조건 이상이 얽히게 만들어야 문제집답다

--------------------------------------------------

[유형 B-2] 두 점 조건으로 계수 결정형
핵심:
- x-a, x-b로 나눈 나머지를 이용해 계수를 정한다.

기본형:
- P(a), P(b) 조건으로 a, b 또는 계수 구하기

구조변형형:
- P(1), P(-1) 형태를 활용
- 상수 2개를 연립해서 구하는 형태
- 마지막에 합, 곱, 차를 묻는 형태

사고확장형:
- 조건 하나를 나머지, 하나를 계수비교로 주는 혼합형
- 성립 여부 판별형
- 조건이 충분한지 판단하는 형태

주의:
- 연립만 풀면 끝나는 뻔한 형태를 줄이고
- 식 구조를 먼저 봐야 하게 설계

--------------------------------------------------

[유형 B-3] 2차식으로 나눌 때 나머지 복원형
핵심:
- P(x)를 (x-a)(x-b)로 나누면 나머지는 1차식 R(x)=mx+n

기본형:
- P(a), P(b)를 이용해 R(x)를 구하시오.

구조변형형:
- R(0), R(1), 특정 값만 묻는 형태
- x^2+x-2처럼 먼저 인수분해해야 하는 형태
- 나머지식 자체가 아니라 계수 합/특정값을 묻는 형태

사고확장형:
- 보기에서 가능한 나머지식 고르기
- 두 조건으로부터 나머지식의 성질 판단
- 2차식 나눗셈과 나머지정리를 함께 쓰는 형태

주의:
- “R(x)=ax+b라 두자” 문장을 계속 반복하지 말 것
- 어떤 경우엔 먼저 인수분해 유도

--------------------------------------------------

[유형 B-4] 나머지와 인수의 관계 해석형
핵심:
- P(a)=0이면 x-a는 인수
- P(a)=P(b) 같은 조건도 구조적으로 해석 가능

기본형:
- P(a)=0 조건으로 인수 여부 판단

구조변형형:
- P(1)=P(-1) 조건에서 공통인수 유도
- P(x)-k가 x-a를 인수로 가지는 조건
- 옳은 식 찾기 객관식

사고확장형:
- 인수정리와 나머지정리 혼합
- 여러 조건 중 어느 것이 충분한지 판단
- “항상 나누어진다” 형태 판별

주의:
- 단순 인수정리 확인형만 반복 금지
- 식 변형이나 발상 유도가 들어가야 함

--------------------------------------------------

[유형 B-5] 조건 판별형
핵심:
- 주어진 조건만으로 값이 정해지는지, 모순인지, 부족한지 판단한다.

기본형:
- 다음 조건으로 P(a)를 구할 수 있는가?

구조변형형:
- 항상 참/항상 거짓/조건 부족 중 판별
- 객관식 5지선다로 선지 소거형 설계
- 주어진 나머지 조건이 서로 양립 가능한지 판단

사고확장형:
- 성립 불가 문항 판별
- 조건 부족을 보완하려면 무엇이 더 필요한지 판단
- 잘못된 풀이 찾기

주의:
- 이 유형은 단원 완성도를 높여줌
- 반드시 몇 문제 넣는 게 좋음

C. 혼합 파트
==================================================

[유형 C-1] 항등식 + 나머지정리 혼합형
핵심:
- 먼저 항등식으로 식을 정리하거나
- 나머지 조건으로 미정계수를 정한 뒤
- 다시 항등식 구조를 활용한다.

기본형:
- P(x)=Q(x)(x-a)+R 형태와 계수비교를 함께 사용

구조변형형:
- 두 나머지 조건 + 계수 조건
- 인수정리 + 항등식
- 나머지식의 특정값 + 전개식 조건

사고확장형:
- 단원 종합 대표문항
- 객관식 심화 또는 서술형 심화
- 풀이 시작 포인트를 바로 잡기 어려운 문항

주의:
- 너무 많으면 지저분해짐
- 대표 심화 1~2문항 정도만 두는 것이 적당

==================================================
D. 실제 생산 규칙
==================================================

각 대표유형마다 문제를 만들 때는 아래 3단계로 만든다.

1단계 기본형
- 교과서/문제집 대표유형처럼 정석적인 문제 1개

2단계 구조변형형
- 같은 개념이지만
  - 조건 순서 변경
  - 묻는 대상 변경
  - 객관식/주관식 전환
  - 판별형 전환
  중 최소 2개 이상 바꾼 문제 1~2개

3단계 사고확장형
- 난이도를 올리되 숫자만 어렵게 하지 말고
  - 조건 추가
  - 역추론
  - 판별
  - 혼합개념
  으로 확장한 문제 1개

==================================================
E. 형님용 추천 스타트 세트
==================================================

항등식과 나머지정리에서 먼저 만들 추천 대표유형 8개:

1. A-1 전개 후 계수비교형
2. A-2 특정값 대입 항등식형
3. A-4 항등식 판별형
4. B-1 나머지정리 직접대입형
5. B-2 두 조건으로 계수 결정형
6. B-3 2차식 나머지 복원형
7. B-5 조건 판별형
8. C-1 혼합 심화형

이 8개만 잘 잡아도
각 유형당 3문항씩만 뽑아도 24문항 풀 가능

F. 금지 규칙
==================================================

- “값을 구하시오”형 남발 금지

- 성립 불가 문항 즉시 폐기
- 정답/해설 불일치 문항 즉시 폐기
