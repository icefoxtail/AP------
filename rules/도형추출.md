# 🚨 [최상위 무결성 규칙 - v6.4 핵심]

## 1️⃣ 단일 SVG 소스 강제 규칙

* SVG는 반드시 한 번만 생성한다

* 생성된 SVG를 단일 원본 문자열(SINGLE SOURCE)로 유지한다

* HTML 출력:
  → SVG 문자열 그대로 사용

* JS 삽입:
  → 동일 SVG 문자열 복사 후
  큰따옴표만 escape 처리

* SVG 재생성, 수정 절대 금지

---

## 2️⃣ 동일성 강제 규칙

* HTML용 SVG와 JS용 SVG는
  좌표 / 라벨 / 스타일 / 수치
  단 하나라도 다르면 실패

---

## 3️⃣ 🚨 삽입 후 재검증 규칙 (핵심 추가)

* JS에 삽입된 SVG를 기준으로
  다시 HTML 구조를 재구성했다고 가정한다

* 이 재구성된 SVG와
  최초 HTML SVG를 비교한다

* 하나라도 다르면:
  → 즉시 실패
  → 출력 금지

---

## 4️⃣ 🚨 출력 고정 규칙

* JS에 삽입된 SVG 문자열을 다시 HTML에 넣었을 때
  최초 HTML SVG와 완전히 동일해야 한다

* 다르면 출력 금지

---

## 5️⃣ 무결성 검증 규칙

* SVG 문자열은 해시 비교 수준으로 동일성 검증했다고 가정

* HTML ≠ JS → 실패

---

## 6️⃣ 금지 사항 (강화)

* Canvas용 / JS용 SVG 따로 생성 금지
* SVG 일부 수정 금지
* 좌표/라벨 일부 변경 금지

# 🎯 [도형 생성 철칙]

1. 모든 수치는 원문 값만 사용 (추측 금지)

2. 반드시 수학적 추론 수행:

   * 좌표 계산
   * 교점 계산
   * 길이/비율 검증

3. 시험지 수준 SVG 생성

4. 필수 요소:

   * viewBox
   * preserveAspectRatio="xMidYMid meet"
   * 비율 유지

5. JS 삽입 시 " → \" 처리

---

# 🎨 [흑백 인쇄 최적화 규칙]

* 색상 의존 금지
* 선 굵기로 구분

메인: stroke-width 2~3
보조: 점선 (3 3)

* 강조:

  * 점 크기 증가
  * 연한 회색 fill 사용

* 텍스트:

  * 10~14px
  * 주요만 bold
