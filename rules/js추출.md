

### **[JS ARCHIVE: 최종 통합 및 변환 엔진 지침]**

**1. ID 글로벌 재색인 및 타입 고정 (Global Re-indexing)**
* **ID 초기화**: 입력 데이터의 기존 `id`를 완전히 무시하고, 배열의 첫 번째 요소부터 **무조건 `1`부터 시작**하여 순차적으로 부여합니다.
* **데이터 타입**: `id` 값은 따옴표가 없는 **순수 숫자(Number)** 타입으로 출력합니다. (예: `"id": 1`)
* **출처 보존**: 원본 PDF의 번호는 `source_metadata.original_id`에 문자열로 반드시 남깁니다.

**2. 발문 및 풀이 원형 보존 (Zero Modification)**
* **텍스트 무결성**: 사용자가 제공한 `content`와 `solution` 필드 내의 **문장 구조, 사용 용어, 설명의 순서, 교육적 의도를 단 한 자도 임의로 수정, 요약, 개선하지 않습니다.**
* **정돈 작업**: 오직 가독성을 위한 줄바꿈(`\\n`) 처리와 수식의 배치 정돈만 수행하며, 원본의 말맛과 풀이 논리를 100% 유지합니다.

**3. JS 표준 래핑 규격 (Mandatory Structure - 예시 준수)**
* 출력물은 반드시 하나의 자바스크립트 코드 블록 내에 아래 선언문을 포함한 구조를 갖춰야 합니다. **(원장님이 주신 표준 예시를 1:1 계승)**
```javascript
window.examTitle = "25_팔마고1_1학기_중간_고1_변형"; // 또는 사용자가 지정한 제목
window.questionBank = [
  {
    "id": 1,
    "source_metadata": {
      "page_number": 32,
      "original_id": "0193"
    },
    "type": "single",
    "level": "하",
    "category": "나머지 정리",
    "content": "다항식 $P(x)=x^{3}-x^{2}+kx+6$이 $x-2$로 나누어떨어질 때, 실수 $k$의 값을 구하시오. [3.5점]",
    "choices": ["$-5$", "$-3$", "$-1$", "$1$", "$3$"],
    "answer": "①",
    "solution": "$P(2)=0 \\\\implies 8-4+2k+6=0 \\\\implies 2k+10=0 \\\\implies k=-5$"
  }
  // ... 이후 문항들 순차 누적
];
```

**4. 수식 이중 이스케이프 및 MathJax (Strict LaTeX)**
* **LaTeX 표기**: 모든 수식은 `$ ... $` 또는 `$$...$$` 형식을 유지합니다.
* **백슬래시 철칙**: 자바스크립트 문자열 내에서 수식이 깨지는 것을 방지하기 위해 모든 백슬래시(`\`)는 반드시 **이중 백슬래시(`\\\\`)**로 치환하여 출력합니다. (예: `\\\\implies`, `\\\\frac`, `\\\\sqrt`)

**5. 필드 종결 및 슬림화 (Field Capping)**
* 각 문항 객체는 **`solution` 필드를 마지막으로 닫으며**, 이후에 `strict_mapping`이나 `integrity_proof` 같은 부차적인 필드는 절대 추가하지 않습니다.
* `type` 필드는 `single`(객관식) 또는 `short`(주관식)로 규격화합니다.
