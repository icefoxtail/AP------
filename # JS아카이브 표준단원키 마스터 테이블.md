# JS아카이브 표준단원키 마스터 테이블

> 기준: 중학교 2022 개정 / 고등학교 2015·2022 개정 교육과정  
> 용도: JS 파일 내 `standardUnitKey`, `standardUnit`, `standardUnitOrder` 필드 작성 기준  
> 관리: 새 단원 추가 시 이 파일에 먼저 등록 후 사용  
> 세부 단원은 RAW- 사용, 추후 편입

## RAW (임시 미분류)

> 대단원에 매핑 안 되는 세부 단원은 `RAW-단원명` 형식으로 임시 사용.  
> 추후 위 체계로 편입. `standardUnitOrder: 999` 사용.

---

## 파일명 컨벤션

```
기출:     YY_학교명_N학기_중간|기말_학년_과목.js
유형:     단원명_학년_유형.js / 단원명_학년_유형2.js
단원평가: 단원명_학년_단원평가.js
쪽지:     단원명_학년_쪽지.js
```

## contentType 값

| 값       | 설명            |
|----------|-----------------|
| 기출     | 학교 시험 기출   |
| 유형     | 유형별 문제 모음 |
| 단원평가 | 단원 마무리 평가 |
| 쪽지     | 쪽지시험         |
