## Lock Rules

- The same `generated/js` file may be edited by only one Codex window.
- The same generated report file may be updated by only one Codex window.
- The same answer source mapping file may be updated by only one Codex window.
- Temporary workstream files must be isolated under `reports/agent-memory` or a separate workstream folder.
