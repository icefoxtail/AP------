# CODEX_ALLOWED_CHANGE_SCOPE

| 작업 유형 | 허용 |
|---|---|
| 문서 작업 | `docs/**`, `CODEX_RESULT.md` 생성/수정 |
| frontend 작업 | 지시된 frontend 파일, 관련 문서 |
| Worker route 작업 | 지시된 route/helper, index 위임, 관련 문서 |
| DB 작업 | 승인된 schema/migration, 관련 문서 |
| 검수 작업 | 파일 읽기, 로컬 검증, 결과 문서 |

허용 범위를 벗어나는 파일은 먼저 사용자 확인을 받는다.
