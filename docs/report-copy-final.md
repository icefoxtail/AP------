# AP Math Report Studio — 학부모 발송 문구 쉬운말 최종본

기준: 학부모가 한 번에 알아듣는 쉬운 말 + 선생님이 실제 쓰는 현장 말투.

원칙:

* 내부 용어는 풀어 쓴다.
* 원인을 지어내지 않는다.
* 없는 정보는 굳이 "없다"고 쓰지 않는다.
* 다음 행동만 말한다.

## 용어 교체표

| 내부/딱딱한 말 | 학부모용 |
| --- | --- |
| 오답 {n}개 | {n}문제 틀렸습니다 |
| 유사문항 / 유사 문제 | 비슷한 문제 |
| 풀이 과정 / 풀이 순서 | 푸는 과정 / 푸는 순서 |
| 조건 표시 | 본문에서는 생략 |
| 개념 연결 / 개념 확인 | 관련 개념 |
| 다시 확인하겠습니다 | 다시 풀어보겠습니다 / 짚겠습니다 / 잡겠습니다 |
| 마무리했습니다 | 잘 봤습니다 / 다 맞았습니다 |
| 보완 / 약점 | 다시 볼 부분 |

## 확정 템플릿

### summary

오답 있음:

```text
이번 시험은 {wrongCount}문제 틀렸습니다. 틀린 문제는 다음 시간에 다시 풀어보겠습니다.
```

만점:

```text
이번 시험은 다 맞았습니다. 다음부터는 조금 더 어려운 문제로 올려보겠습니다.
```

### trend

첫 시험:

```text
이번이 첫 시험입니다. 다음 시험부터 점수 변화를 같이 보여드리겠습니다.
```

추이:

```text
최근 {selected.length}번 시험은 {올라가는/내려가는} 중입니다. 가장 높은 점수 {bestScore}점, 낮은 점수 {worstScore}점입니다.
```

증감:

```text
첫 시험보다 {absScoreDelta}점 올랐습니다.
첫 시험보다 {absScoreDelta}점 낮아졌습니다.
첫 시험과 같은 점수입니다.
```

### weakness

만점 또는 틀린 문제가 없으면 블록을 표시하지 않는다.

```text
여러 번 틀린 문제부터 다시 풀어보겠습니다.
{priorityText} 먼저 다시 풀고, 비슷한 문제로 한 번 더 보겠습니다.
다음 시간에 틀린 문제를 한 번 더 짚고 넘어가겠습니다.
```

### plan

```text
다음 시간에 틀린 문제를 다시 풀고, 비슷한 문제로 한 번 더 연습하겠습니다.
어려웠던 문제는 푸는 순서와 관련 개념을 다시 짚겠습니다.
맞힌 유형은 그대로 가고, 조금 더 어려운 문제로 올려보겠습니다.
{wrongNums}번을 다시 풀어보고, 비슷한 문제까지 잡겠습니다.
```

### teacherOpinion

```text
이번에 틀린 문제를 기준으로, 다음 시간에 다시 풀어보겠습니다.
이번 시험은 다 맞았습니다. 특히 {excellentNums}번처럼 어려웠던 문제도 잘 풀었습니다.
어려웠던 문제는 관련 개념을 다시 짚겠습니다.
계속 틀리는 부분과 이번에 새로 틀린 부분을 나누어, 다음 시간에 보겠습니다.
선생님 메모는 다음 수업에 반영하겠습니다: {teacherMemo}
```

### parentMessage

오답 있음:

```text
안녕하세요, AP수학입니다.

이번 리포트에는 점수와 함께 틀린 문제, 문제 난이도를 정리했습니다.
다음 시간에 틀린 문제를 다시 풀고, 같은 실수가 나오지 않게 비슷한 문제로 한 번 더 연습하겠습니다.
가정에서는 문제를 다 푼 뒤 한 번 더 검토하는 습관만 봐주시면 됩니다.
```

만점:

```text
{studentName} 학생은 이번 시험에서 다 맞았습니다.
다음 시간에는 다음 단원과 조금 더 어려운 문제로 올려보겠습니다.
가정에서는 지금처럼 봐주시면 됩니다.
```

### kakaoSummary

오답 있음:

```text
안녕하세요, AP수학입니다. {student.name} 학생의 「{exam_title}」 결과를 안내드립니다.
이번 시험은 {score}점입니다. 자세한 내용은 PDF 리포트에 정리했습니다.
틀린 문제는 {priorityText} 위주로 다음 시간에 다시 풀고, 비슷한 문제까지 잡겠습니다.
```

만점:

```text
안녕하세요, AP수학입니다. {student.name} 학생의 「{exam_title}」 결과를 안내드립니다.
이번 시험은 {score}점으로, 다 맞았습니다.
다음 시간에는 조금 더 어려운 문제로 올려보겠습니다. 자세한 내용은 PDF 리포트에 있습니다.
```
