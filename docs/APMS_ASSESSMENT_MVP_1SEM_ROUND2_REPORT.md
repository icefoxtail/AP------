# APMS 시험지 보관함 1학기 MVP Round 2 보고서

## 1. 이번 보정 핵심
- 중3 1학기 기말 원천을 추가 반영했습니다.
- 정확 경로가 존재하는 그래프/이미지 문항은 평가팩에 포함했습니다.
- 한 평가지 안에서는 같은 원문 문항과 같은 유형을 중복하지 않도록 구성했습니다.
- 평가팩끼리 좋은 문항을 재사용하는 것은 허용했습니다.

## 2. 평가팩 수
- 진단평가: 5개
- 단원평가: 43개
- 중간·기말평가: 8개

## 3. 새로 추가한 중3 기말 평가팩
- [단원평가] 중3 1학기 기말 다항식의 곱셈과 인수분해 단원평가 / 10문항 / 이미지 포함 / 유형 10종
- [단원평가] 중3 1학기 기말 이차방정식 단원평가 / 15문항 / 이미지 포함 / 유형 15종
- [단원평가] 중3 1학기 기말 이차함수와 그래프 단원평가 / 15문항 / 이미지 포함 / 유형 15종
- [중간·기말평가] 중3 1학기 기말 대비팩 / 25문항 / 이미지 포함 / 유형 25종

## 4. 중3 기말 사용 가능 문항
- 원천 문항: 262문항
- 이미지 경로 보정 후 사용 가능 문항: 260문항
