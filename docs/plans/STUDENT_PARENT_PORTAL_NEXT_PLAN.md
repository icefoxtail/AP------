# STUDENT_PARENT_PORTAL_NEXT_PLAN

## 1. 현재 상태

학생 포털, 플래너, OMR, 숙제 사진, material OMR 흐름이 존재한다. 학부모 전용 포털은 1차 확인에서 명확히 특정하지 못했다.

## 2. 최종 목표

학생/학부모 포털을 출결, 숙제, 리포트, 청구, 공지, 상담으로 확장하되 AP Math OMR 금지 정책을 보존한다.

## 3. 절대 금지/보류

- 학생 시험지 직접 열기 금지
- 일반 OMR 제출 완료 후 수정 금지
- 학부모/public 경로에 운영자 메모/수납 내부 로그 노출 금지

## 4. Phase 구조
