[작업 내용]
업로드된 각 JS 파일에 대해 아래 처리를 해줘:

1. 파일 전체 SHA-256 해시 계산
2. 파일 상단에 봉인 주석 블록 추가
   (검수완료, 검수일, 문항 수, SHA-256)
3. questionBank 첫 줄 위에 examTitle, examMeta(Object.freeze) 추가
   - examTitle은 파일명에서 추출 (확장자 제외)
   - examMeta에 reviewed/reviewedAt/sha256/count 포함
4. 파일 맨 끝에 Object.freeze(window.questionBank) 추가
5. 출력 파일명은 원본 파일명 확장자 앞에 _r 추가
   예) 24_왕운중3_1학기_중간_중3.js → 24_왕운중3_1학기_중간_중3_r.js
6. 처리된 파일 전부 outputs에 저장 후 다운로드 링크 제공

봉인 형식은 24_왕운중3_1학기_중간_중3.js 파일과 동일하게 맞춰줘.
