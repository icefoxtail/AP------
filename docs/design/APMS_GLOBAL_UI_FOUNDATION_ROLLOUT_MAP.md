# APMS Global UI Foundation Rollout Map

## 적용 순서

1. classroom
2. student portal
3. 상담
4. 교재
5. 수업자료
6. 성적
7. OMR
8. 시간표
9. 출석부
10. 리포트

## 단계별 검토 기준

각 단계는 기존 문구/버튼명/화면명 보존, API 호출 보존, 전역 selector 미수정, card 안 card 감소, line row 적용, node syntax check를 확인한다.

시간표, 출석부, 리포트는 출력/표/색상 의존성이 커서 마지막에 별도 검수한다.
