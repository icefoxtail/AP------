# APMS Global UI Foundation Student Portal Plan

## 후속 적용 기준

Student portal은 classroom 다음 phase에서 적용한다. 이번 작업에서 `apmath/student/index.html`은 수정하지 않는다.

## 원칙

- 학생에게 필요한 정보만 버튼처럼 보이게 한다.
- 제출완료/미제출 상태를 색상으로 과하게 강조하지 않는다.
- 완료 상태는 조용한 텍스트와 chip으로 표현한다.
- 학생 화면은 더 단순하고 더 명확해야 한다.

## 보류

학생 포털의 문구, 버튼명, OMR 흐름, 제출 상태 로직은 이번 작업에서 건드리지 않는다.
