# APMS Global UI Foundation Dashboard Bridge

## 현재 위치

`dashboard-foundation.css`는 dashboard 특수 화면 CSS로 유지한다. 이번 작업에서 `dashboard.js`와 dashboard CSS를 수정하지 않는다.

## 연결 기준

- 오늘일지 outer card는 유지한다.
- 일지 row는 line row 규칙을 따른다.
- 원장/선생님 카드와 상담 병합 기준은 기존 문서와 로직을 보존한다.
- dashboard는 global foundation의 token과 row 철학을 따르되, 실제 적용은 별도 phase에서 수행한다.

## 금지

이번 작업에서 dashboard 화면 구조, 버튼명, 상담 병합 로직, 관리자/원장 노출 상태를 바꾸지 않는다.
