# Timetable Attendance Classroom SOP

Use for timetable, attendance, classroom, homework, and class daily work.

## Protected Areas

- `apmath/js/timetable.js`
- `apmath/js/classroom.js`
- `docs/timetable`
- timetable version/conflict routes
- attendance-homework routes
- class-daily routes

## Procedure

1. Identify whether the task touches operating, draft, staging, active, scheduled, or cancelled timetable state.
2. Protect teacher field workflows.
3. Preserve existing class/attendance/homework language.
4. Do not merge timetable states casually.
5. Verify conflict/version logic when applicable.
