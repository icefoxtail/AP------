# Public Site Brand SOP

Use for public site, manual, brand naming, Wangji, AP Math, CMath, and EIE separation.

## Protected Areas

- root `index.html`
- `manual`
- `alive`
- `rules`
- brand/domain plan docs

## Procedure

1. Preserve existing brand names unless the task explicitly changes them.
2. Keep AP Math, Wangji, CMath, and EIE separated.
3. Do not rename UI labels or public wording as cleanup.
4. For public pages, verify that content is intended for external users.
