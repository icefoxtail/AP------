# Archive Exams SOP

Use for exam source data and image asset work.

## Protected Areas

- `archive/exams`
- `archive/assets/images`

## Procedure

1. Confirm the exact exam files allowed by the task.
2. Avoid broad recursive rewrites.
3. Preserve image paths and question identifiers.
4. Validate parse/syntax for changed JS data files.
5. Record any skipped large-asset checks.

## Default Lock

Inventory and sampling are allowed for report tasks. Edits require explicit exam-data scope.
