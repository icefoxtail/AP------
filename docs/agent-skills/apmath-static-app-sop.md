# AP Math Static App SOP

Use for `apmath/` static UI work.

## Protected Areas

- `apmath/js`
- `apmath/index.html`
- `apmath/student`
- `apmath/planner`
- `apmath/homework`
- UI text, menu names, mode names, button labels, and visible foundation exposure.

## Procedure

1. Read the AP rulebook and domain index.
2. Identify the exact screen and JS module.
3. Preserve vanilla JS patterns and existing UI language.
4. Do not expose hidden foundation functionality without explicit user approval.
5. Run syntax or browser verification required by the task.

## Default Lock

No broad redesign, route/menu/mode restructure, or hidden dashboard exposure by default.
