# CODEX_TASK.md

## 0. 작업 목표

수업자료 화면에서 지시하지 않은 위험 버튼을 제거하고, `복습 출력` 표현을 `오답번호 출력`으로 변경한다.

이번 작업은 화면 문구/버튼 제거만 한다.

## 3. 수정 1 — 자료 등록 화면의 비활성화 UI 제거

`apmath/js/study-material-wrong.js`의 `materialForm()`에서 등록된 자료 카드에 표시되는 `비활성화` 버튼을 제거한다.

현재 예시:

```js
<button class="btn" onclick="disableStudyMaterial('${h(m.id)}')">비활성화</button>
````

수정 후 자료 카드 안에는 어떤 관리 버튼도 없어야 한다.

---

## 4. 수정 2 — 자료 상태 표시 제거

자료 카드의 meta 표시에서 상태값을 제거한다.

아래 표현이 자료 등록 화면에 나오면 안 된다.

```text
사용 중
사용 안 함
삭제됨
active
inactive
deleted
```

현재 예시:

```text
문제기본서 · 중3 · 1학기 · 1-1078 · 사용 안 함
```

수정 후:

```text
문제기본서 · 중3 · 1학기 · 1-1078
```

DB 값이나 API는 건드리지 않는다. 화면 표시만 제거한다.

---

## 5. 수정 3 — `복습 출력`을 `오답번호 출력`으로 변경

수업자료 화면 상단 탭명에서 다음 문구를 변경한다.

```text
복습 출력 → 오답번호 출력
```

적용 위치:

* 상단 탭명
* 해당 탭의 버튼/안내 문구 중 `복습표`, `복습 출력`, `복습 방법`, `복습표 보기` 표현

---

## 6. 수정 4 — 오답번호 출력 탭 문구 정리

`reviewForm()`과 `renderReviewText()`에서 사용자에게 보이는 문구를 아래처럼 정리한다.

### 6.1 탭명

```text
오답번호 출력
```

### 6.2 버튼명

기존:

```text
복습표 보기
```

변경:

```text
오답번호 보기
```

### 6.3 기본 안내 문구

기존:

```text
조건을 선택하세요.
```

유지 가능.

단, `복습표`, `복습 출력`, `복습 지시표`라는 표현은 나오지 않게 한다.

### 6.4 출력 제목

기존 예시:

```text
홍길동 복습표
```

변경:

```text
홍길동 오답번호
```

### 6.5 하단 안내

기존:

```text
복습 방법
1. 위 번호만 다시 풀기
2. 풀이 과정 표시하기
3. 다음 수업 때 확인
```

변경:

```text
확인 방법
1. 위 번호만 다시 풀기
2. 풀이 과정 표시하기
3. 다음 수업 때 확인
```

```

### 9.3 오답번호 출력 탭

오답번호 보기
```

```text
홍길동 오답번호
```

하단 안내 제목은 아래처럼 보여야 한다.

```text
확인 방법
```
