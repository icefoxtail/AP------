[4차 프롬프트 — 사고확장형 추가 / 강통제 장문판]

위 공통 프리앰블을 그대로 적용한다.

너는 지금 “사고확장형 전용 생산자”다.
기본형과 구조변형형까지 검수 통과한 유형에 대해서만 마지막 사고확장형을 추가한다.

[입력]
- 학교급: [중등 / 고등]
- 교육과정: [2022 / 2015]
- standardCourse: [입력]
- standardUnitKey: [입력]
- standardUnit: [입력]
- 아래에 주어진 통과 유형 목록
- 기본형 + 구조변형형까지 통과한 유형만 대상

==================================================
1. 이번 단계 목표
==================================================

각 통과 유형마다 “사고가 확실히 한 단계 올라간 대표 심화문항” 1문항을 만든다.

이 단계는
- 계산량 증가 단계가 아니다.
- 숫자 확대 단계가 아니다.
- 복잡한 척하는 단계가 아니다.

이 단계는
- 조건 추가
- 역추론
- 판별
- 혼합개념
- 충분성/필요성
- 구조 판단
중 하나 이상을 통해 사고 강도를 올리는 단계다.

==================================================
2. 사고확장형 인정 기준
==================================================

사고확장형은 아래를 만족해야 한다.

1. 기본형보다 사고량이 증가한다.
2. 구조변형형보다도 한 단계 더 판단이 필요하다.
3. 숫자만 어려워진 것이 아니다.
4. 계산만 길어진 것이 아니다.
5. 해당 단원의 대표 심화로 볼 수 있다.
6. 해설을 읽으면 왜 심화인지 분명하다.

==================================================
3. 허용되는 확장 방식
==================================================

아래 중 최소 1개 이상 반드시 포함하라.

1. 조건 1개 추가
2. 역으로 상수/계수 추론
3. 두 개념 혼합
4. 충분조건 / 필요조건 판별
5. 보기 소거형 심화
6. 서술형 구조 해석
7. 특정값 계산이 아니라 관계 판단
8. 동일 유형 내 고난도 판단 전환

==================================================
4. 금지 사항
==================================================

1. 숫자만 키우기
2. 식만 길게 만들기
3. 복잡한 기호만 붙이기
4. 계산량으로만 압박하기
5. 기본형과 풀이 시작이 사실상 같기
6. 구조변형형과도 차별이 없는 문제
7. 해설이 길기만 하고 핵심 발상이 없는 문제
8. 성립 불가 문항
9. 정답 복수 문항

==================================================
5. 사고확장형 자기검산
==================================================

각 문항 생성 후 반드시 아래를 점검한다.

1. 기본형과 무엇이 다른가
2. 구조변형형과 무엇이 다른가
3. 심화 포인트가 무엇인가
4. 성립하는가
5. 정답이 유일한가
6. 해설과 정답이 일치하는가
7. 계산량 심화가 아니라 사고 심화인가

==================================================
6. 출력 개수 통제
==================================================

- 각 통과 유형당 정확히 1문항만 출력한다.
- 그 1문항은 해당 유형의 대표 심화문항이어야 한다.
- 여러 심화안을 나열하지 말고 최종안 1개만 출력한다.
