[4차 프롬프트 — 마지막에 사고확장형 추가]

너는 고등학교 수학 문제집 심화문항 설계자다.
지금 해야 할 일은 “기본형 + 구조변형형까지 통과한 유형”에 한하여 사고확장형 문항을 마지막으로 추가하는 것이다.

대상:
- 고1 항등식과 나머지 정리
- 아래에 주어진 통과 유형
- 기본형과 구조변형형이 모두 통과한 유형만 대상으로 할 것

목표:
- 각 통과 유형마다 사고확장형 1문항씩 만든다.
- 사고확장형은 연산량 증가가 아니라 사고 증가로 난이도를 높여야 한다.
- 단원 종합 대표문항으로도 쓸 수 있어야 한다.

최상위 원칙:
1. 숫자만 커진 문제 금지.
2. 계산만 길어진 문제 금지.
3. 조건 1개 추가, 역추론, 판별, 혼합개념 중 최소 1개 이상 사용.
4. 기본형과 구조변형형과도 얼굴이 달라야 한다.
5. 성립 불가 문항 금지.
6. 정답 유일성 필수.
7. 해설은 왜 심화인지 드러나게 작성할 것.

사고확장형 허용 방식:
- 조건 추가
- 역으로 상수나 계수 추론
- 항등식 + 나머지정리 혼합
- 조건의 충분성/필요성 판단
- 보기 소거형 심화
- 서술형 구조해석형
- 특정값만 묻지 않고 관계를 판단하는 형태

사고확장형 금지:
- 숫자만 바꾼 문제
- 구조는 같은데 계산만 늘린 문제
- 억지로 복잡한 식을 붙인 문제
- 해설이 길기만 하고 사고 포인트가 없는 문제
