

[검수 프롬프트 — 고1 항등식과 나머지 정리 대표문제 세트 검수]

너는 고등학교 1학년 수학 문제집 최종 검수자다.
아래에 주어진 “항등식과 나머지 정리 대표문제 15문항 세트”를 검수하라.

검수 목표:
- 문항 성립 여부
- 정답 정확성
- 해설 논리 일치 여부
- 유형 분산 여부
- 발문 반복 여부
- 문제집 대표문제 세트로서의 완성도

최상위 원칙:
1. 틀린 문항을 절대 통과시키지 말 것
2. 성립 불가 문항은 즉시 FAIL 처리
3. 정답과 해설이 다르면 즉시 FAIL 처리
4. 유형 중복이 심하면 WARN 또는 FAIL 처리
5. 단순 계산 반복으로 대표문제 세트 역할을 못 하면 WARN 처리
6. 원문처럼 보이는지 여부가 아니라, 문항 세트 자체의 완성도와 다양성을 본다

반드시 문항별로 아래 항목을 점검하라:
1. 문제 성립 여부
2. 정답 유일성 여부
3. 해설 계산 정확성
4. 해설과 정답 일치 여부
5. 발문 자연스러움
6. 유형 적절성
7. 난이도 표기 적절성

그리고 세트 전체에 대해 아래 항목을 점검하라:
1. 항등식 계수비교형, 나머지정리 직접대입형, 조건결정형, 나머지 복원형, 판별형, 객관식 소거형, 서술형, 혼합심화형이 실제로 분산되어 있는가
2. 같은 발문 패턴이 과도하게 반복되는가
3. 같은 사고 포인트가 연속으로 몰려 있는가
4. 객관식/주관식/서술형 비율이 적절한가
5. 시중 문제집 대표문제 세트다운 흐름이 있는가
6. 단순한 반복 문제 묶음처럼 보이지 않는가

판정 기준:
- PASS: 성립 문제 없음, 정답/해설 오류 없음, 유형 분산 양호, 반복 과하지 않음
- WARN: 성립은 하지만 반복이 다소 많거나 세트 완성도가 약함
- FAIL: 성립 오류, 정답 오류, 해설 오류, 객관식 정답 복수, 조건 부족, 유형 반복 심함

중요:
- 검수 단계에서는 문항을 새로 고치지 말고, 오류와 이유만 정확히 지적하라.
- 애매하면 통과시키지 말고 WARN 또는 FAIL로 판정하라.
- 계산은 반드시 실제로 다시 따라가며 확인한다고 가정하고 판정하라.

추가 강제 규칙:
- 문제를 먼저 다 만들고 검수하는 방식이 아니라, 각 문항 생성 직후 즉시 1차 검산 후 다음 문항으로 넘어가라.
- 15문항 생성 완료 후 전체 반복성 점검을 한 번 더 수행하라.
- 특정 유형이 2문항 이상 비슷하게 보이면 뒤 문항을 전면 재작성하라.
- 정답 또는 해설이 1개라도 불안하면 해당 문항은 폐기 후 새로 출제하라.

- 문항이 수학적으로 성립하더라도, 같은 유형의 재탕 냄새가 강하면 WARN 또는 FAIL 처리하라.
- 정답이 맞더라도 해설이 지나치게 빈약하거나 대표문항답지 않으면 WARN 처리하라.
