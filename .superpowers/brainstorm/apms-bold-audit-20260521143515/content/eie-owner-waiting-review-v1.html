등록 대기

의미: 아직 정식 재원으로 확정하지 않은 학생입니다. 상담 후 등록 예정, 반 배정 전, 연락 후 결정 같은 상태를 담습니다.

호버: 등록 대기 합계와, 학년 데이터가 있으면 학년별 수. 없으면 학년 미분류.

클릭: 학생 관리 화면을 대기 상태로 필터링한 목록 또는 대기 학생 모달.

시간표 확인

의미: 엑셀/시간표에서 가져온 데이터 중 사람이 확인해야 하는 항목입니다. 예: 학생 이름 후보가 애매함, 수업 셀 상태가 확인 필요, 배정 정보가 충돌함.

호버: 전체 확인 대상 수와 유형별 수. 예: 학생 후보, 시간표 셀, 데이터 오류.

클릭: 시간표 관리 또는 학생 후보 검토 화면으로 이동해서 바로 처리.
