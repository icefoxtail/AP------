1. 수정/신규 파일 목록
- 수정: archive/index.html
- 수정: archive/mixer.html

2. 구현 완료 기능
- `archive/index.html` 카드 파일명 표시를 마지막 파일명 기반의 보기 좋은 텍스트로 변경
- `archive/mixer.html` 시험지 목록 파일명 표시를 마지막 파일명 기반의 보기 좋은 텍스트로 변경
- 실제 `ex.file`, `paper.file`, 엔진/믹서 전달 경로와 버튼 동작은 유지

3. 검사 결과
- 별도 자동 검사 명령 없음

4. 수동 테스트 체크리스트
- index 카드에 폴더 경로가 보이지 않는지 확인
- mixer 시험지 목록에도 폴더 경로가 보이지 않는지 확인
- 시험 버튼 정상 동작 확인
- 해설 버튼 정상 동작 확인
- 정답 버튼 정상 동작 확인
- 믹서에서 시험지 선택 정상 확인
- 믹서에서 문항 로드 정상 확인
- mixed_engine 출력 정상 확인

5. 미구현/주의 항목
- 표시 텍스트만 변경했으며 실제 파일 경로 값과 로딩 로직은 그대로 유지
