## 2. 구현 완료
- 상단 담임 열에서 `Foreigner`를 제거했다.
- `Foreigner`는 오른쪽 미니 클래스룸의 요일별 담당 선택 목록에는 계속 남겨두었다.
- `Foreigner`, `Forei`, `Foreign`, `원어민`, `외국인` 계열 값은 담임 열 기준에서 제외하고, 담임이 없는 수업은 `미정` 열로 들어가게 했다.
- 오른쪽 미니 클래스룸이 열려도 왼쪽 시간표 카드가 찌그러지지 않도록 담임 열 최소폭과 가로 스크롤 구조를 재계산했다.
- 시간표 전체 폭을 더 쓰도록 `eie-main`, `eie-v2-layout`, `eie-v2-board-grid` 폭 계산을 재정리했다.
- 선택 전에는 시간표가 전체 폭을 쓰고, 선택 후에는 고정 폭 오른쪽 패널과 왼쪽 가로 스크롤 시간표가 균형 있게 배치되도록 조정했다.
- 담임 열은 남은 화면 폭이 충분하면 자동으로 균등 확장되고, 부족하면 최소폭을 유지한 채 가로 스크롤된다.

- 더 좁은 화면에서는 담임/카드 폭을 줄여 깨뜨리지 않고, 가로 스크롤로 넘겨 보게 했다.
